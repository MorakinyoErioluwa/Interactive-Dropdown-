# Interactive Dropdown Navigation

This is a solution to the Interactive Dropdown Navigation challenge from Frontend Mentor.

## 📌 Overview

A responsive navigation component with interactive dropdown menus built using HTML and Tailwind CSS . The project focuses on user interaction, clean layout structure, and mobile-first design.

## 🚀 Features

- Interactive dropdown menus
- Responsive navigation for all screen sizes
- Mobile hamburger menu
- Smooth open/close animations
- Clean and accessible UI structure

## 🛠️ Built With

- HTML5
-Tailwind CSS3
